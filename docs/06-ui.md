## 페이지 별 UI 구조 생성 및 업데이트

-   `src/App.css`

    -   전체 화면 레이아웃 및 중앙 정렬을 위한 스타일 조정
    -   배경색 설정 및 카드 영역 스타일 정리
    -   버튼 비활성화 상태 스타일 정의
