# API 요청 구현

## 1. Open Trivia DB API 연동

-   Open Trivia DB를 사용하여 퀴즈 문제 데이터를 가져오는 API 요청 구현
-   True / False 타입 퀴즈 문제만 사용하도록 요청 파라미터 구성
-   문제 개수와 난이도를 인자로 받아 유연하게 요청 가능하도록 설계

---

## 2. axios 기반 비동기 요청 처리

-   axios 라이브러리를 사용하여 HTTP GET 요청 처리
-   async / await 문법을 사용한 비동기 흐름 구성
-   API 응답 데이터에서 문제 목록(results)만 추출하여 반환

---

## 3. 서비스 레이어 분리

-   API 요청 로직을 `services/fetchQuestions` 파일로 분리
-   App 컴포넌트에서는 API 세부 구현을 알 필요 없이 함수만 호출
-   데이터 통신과 UI 로직을 명확히 분리한 구조 유지

---

## 4. 타입과 API 응답 연결

-   API 응답 구조에 맞춰 `IQuestion` 타입 정의
-   API 요청 함수의 반환 타입으로 `IQuestion[]` 사용
-   이후 상태 관리 및 컴포넌트 렌더링에서 타입 안정성 확보

---

## 5. App 컴포넌트와 API 연결

-   앱 초기 로딩 시 퀴즈 문제 API 요청 수행
-   응답 데이터를 상태로 저장하여 퀴즈 진행 로직에 사용
-   API 요청 완료 후 로딩 상태 해제하여 화면 전환 처리

---

## 요약

-   퀴즈 문제 데이터를 가져오기 위한 API 요청 로직 구현 완료
-   axios + 서비스 레이어 구조로 API 연동 정리
-   타입 기반으로 API 응답을 안전하게 앱 로직에 연결
